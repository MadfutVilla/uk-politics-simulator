UK Politics Simulator

How to open locally:
1. Extract this ZIP.
2. Double-click index.html.

How to put online:
1. Go to https://vercel.com or https://netlify.com
2. Create a free account.
3. Drag the extracted folder into the deploy box.
4. The site will publish and give you a public link.

Files:
- index.html
- styles.css
- script.js
